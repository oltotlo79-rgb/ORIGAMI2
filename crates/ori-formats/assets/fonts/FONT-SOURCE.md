The unmodified upstream font is bundled so that Japanese and English
instruction text can be rendered without relying on fonts installed on the
user's Windows or macOS system. The generated PDF converts glyphs to vector
outlines. The SVG image archive includes the font and its SIL Open Font License
so extracted SVG pages have a local, redistributable font source.

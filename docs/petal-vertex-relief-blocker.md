# 花弁折りの正厚頂点リリーフ境界

Kawasaki・Maekawaを満たす次数4内点では、選択した3本と残る補助1本からなる限定4候補で、三段の厳密な閉鎖端点を構成できる。

一方、現在の材料モデルは共通内点に集まる正厚プリズムの頂点リリーフを表現しない。このため初期の平坦姿勢が正厚衝突検証を通らず、花弁折り権限は発行されない。接触を例外扱いして証明を迂回してはならない。

本番route/UIは閉じたままとし、次の全条件を満たすnative材料モデルが導入されるまでunsupportedとする。

- 頂点リリーフ形状と寸法がプロジェクト材料へ永続化される。
- 静的・連続・層輸送の全証明が同一リリーフinstanceへ束縛される。
- リリーフなし、寸法不一致、途中段失敗ではrecord・slot・projectを変更しない。
